- **报名提交时间：** 2025年12月11日 00:00:00 — 2026年01月20日 23:59:59

## 参赛要求

### 基本要求
- 单人参赛（不支持组队）
- 完成天池平台实名认证
- 每人可提交多个作品，每个作品独立评分

### 有效作品定义
1. 在阿里云 ESA Pages 平台部署应用
2. 提交可访问的 URL
3. 提供公开的 GitHub 源码仓库

### 技术栈
- 支持 React、Vue 等主流前端框架
- 鼓励使用 AI 生成内容

作品从三个方向独立评分，奖金可叠加：
